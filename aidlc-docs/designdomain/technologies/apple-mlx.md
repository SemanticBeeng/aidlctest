# Technology: Apple MLX

## Classification
- Primary: [[Model Inference Engine]] (Apple/Metal runtime)

## What it is (in this architecture)
MLX is an Apple/Metal-native tensor framework used for Mac/iOS ecosystem experimentation and prototyping.

## When to choose it
- Apple-hardware-first experimentation/prototyping
- You want to explore Apple-native runtime paths

## Evaluation notes
- Typically used in-process; if used for evaluation it behaves like an in-process inference engine.
